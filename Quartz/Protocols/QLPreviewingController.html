- (void)preparePreviewOfSearchableItemWithIdentifier:(NSString *)identifier queryString:(NSString *)queryString completionHandler:(QLPreviewItemLoadingBlock)handler

Parameters

identifier

The identifier of the CSSearchableItem the user interacted with in Spotlight.
handler

The completion handler should be called whenever the view is ready to be displayed. A loading spinner will be shown until the handler is called. It can be called asynchronously after the method has returned.

Discussion

@abstract Use this method to prepare the content of the view controller with the data that the searchable item represents.

@discussion This method will be called only once. It will be called in the main thread before presenting the view controller. Heavy work potentially blocking the main thread should be avoided in this method.
