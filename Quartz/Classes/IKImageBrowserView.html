– dropOperation

Returns the current dropOperation. The returned value is valid when a drop occurred and until next drop.

- (IKImageBrowserDropOperation)dropOperation

Discussion

@method dropOperation

Returns IKImageBrowserDropOn if the drop occurs on an item. Returns IKImageBrowserDropBefore otherwise. In drag and drop, used to specify a dropOperation. For example, given a browser with N cells (numbered with cell 0 at the top left visually), a cell of N-1 and operation of IKImageBrowserDropOn would specify a drop on the last cell. To specify a drop after the last cell, one would use an index of N and IKImageBrowserDropBefore for the operation.

– setDropIndex:dropOperation:

Used if you wish to ÒretargetÓ the proposed drop.

- (void)setDropIndex:(NSInteger)index dropOperation:(IKImageBrowserDropOperation)operation

Discussion

@method setDropIndex:dropOperation:

To specify a drop on the second item, one would specify index as 1, and operation as IKImageBrowserDropOn. To specify a drop after the last item, one would specify index as the number of items and operation as IKImageBrowserDropBefore. Passing a value of Ð1 for index, and IKImageBrowserDropOn as the operation causes the entire browser view to be highlighted rather than a specific item. This is useful if the data displayed by the receiver does not allow the user to drop items at a specific item location.
