Overview

@class CBMutableDescriptor

@discussion Used to create a local characteristic descriptor, which can be added to the local database via CBPeripheralManager. Once a descriptor is published, it is cached and can no longer be changed. Descriptor types are detailed in @link CBUUID.h @/link, but only the Characteristic User Description and Characteristic Presentation Format descriptors are currently supported. The Characteristic Extended Properties and Client Characteristic Configuration descriptors will be created automatically upon publication of the parent service, depending on the properties of the characteristic itself.
