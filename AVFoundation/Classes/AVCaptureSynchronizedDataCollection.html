@class AVCaptureSynchronizedDataCollection @abstract A collection of AVCaptureSynchronizedData objects.

@discussion AVCaptureDataOutputSynchronizer’s -dataOutputSynchronizer:didOutputSynchronizedDataCollection: delegate method delivers a collection of AVCaptureSynchronizedData objects which can be iterated by use AVCaptureOutput. AVCaptureSynchronizedDataCollection supports object subscripting and fast enumeration of the data outputs as keys.
