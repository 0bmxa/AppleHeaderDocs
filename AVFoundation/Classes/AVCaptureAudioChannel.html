AVCaptureAudioChannel Class Reference

Inherits from NSObject
Declared in AVCaptureSession.h

Overview

@class AVCaptureAudioChannel @abstract AVCaptureAudioChannel represents a single channel of audio flowing through an AVCaptureSession.

@discussion An AVCaptureConnection from an input producing audio to an output receiving audio exposes an array of AVCaptureAudioChannel objects, one for each channel of audio available. Iterating through these audio channel objects, a client may poll for audio levels. Instances of AVCaptureAudioChannel cannot be created directly.
