+ textStyleRuleWithTextMarkupAttributes:textSelector:

Creates an instance of AVTextStyleRule with the specified text markup attributes and an identifier for the range or ranges of text to which the attributes should be applied. @param textMarkupAttributes An NSDictionary with keys representing text style attributes that are specifiable in text markup. Eligible identifiers are determined by the format and content of the legible media. A value of nil indicates that the textMarkupAttributes should be applied as default styles for all text unless overridden by content markup or other applicable text selectors.
