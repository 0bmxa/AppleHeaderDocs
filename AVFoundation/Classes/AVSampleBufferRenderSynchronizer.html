– addPeriodicTimeObserverForInterval:queue:usingBlock:

Requests invocation of a block during rendering to report changing time. @param interval The interval of invocation of the block during normal rendering, according to progress of the current time of the timebase. @param queue The serial queue onto which block should be enqueued. If you pass NULL, the main queue (obtained using dispatch_get_main_queue()) will be used. Passing a concurrent queue to this method will result in undefined behavior. @param block The block to be invoked periodically. @result An object conforming to the NSObject protocol. You must retain this returned value as long as you want the time observer to be invoked by the synchronizer. Pass this object to removeTimeObserver: to cancel time observation.
