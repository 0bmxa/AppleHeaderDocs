+ textureNoiseWithSmoothness:size:grayscale:

Create a texture containing colored noise. The noise texture is tileable with itself.

+ (instancetype)textureNoiseWithSmoothness:(CGFloat)smoothness size:(CGSize)size grayscale:(BOOL)grayscale

Parameters

smoothness

how similar neighboring pixels are. A value of zero is like static, one is smooth.
grayscale

if YES, RGB and A will all be the same. If no, RGB and A will all be different. A is not pre-multiplied, because the intent is that if you read a texel in a shader, all four values will be exactly the same value if grayscale, or four different, uncorrelated values if not grayscale.
rect

the size of the resulting texture.

Discussion

Create a texture containing colored noise. The noise texture is tileable with itself.
