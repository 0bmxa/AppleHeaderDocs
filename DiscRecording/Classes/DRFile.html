Overview

@class      DRFile

A file can be either a pointer to an exiting file (residing on a hard drive for example) or can be created at burn time from data passed into the file object as requested. DRFiles can only exist inside of virtual @link //apple_ref/occ/cl/DRFolder DRFolder @/link objects.

@category       DRFile (VirtualFiles)

This category on DRFile defines methods that allow the file to be specified using data passed in at creation time.

@category       DRFile (VirtualLinks)

This category on DRFile defines methods that allow various link/alias files to be created on the resulting disc which may not exist in the source.
