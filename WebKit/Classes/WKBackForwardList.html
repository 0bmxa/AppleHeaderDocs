Returns the item at a specified distance from the current item.

- (nullable WKBackForwardListItem *)itemAtIndex:(NSInteger)index

Parameters

index

Index of the desired list item relative to the current item: 0 for the current item, -1 for the immediately preceding item, 1 for the immediately following item, and so on.

Return Value

The item at the specified distance from the current item, or nil if the index parameter exceeds the limits of the list.
