Parameters

An invitation message. If nil, Messages uses a default invitation message.
roomName

The name of the chat room.
handle

The handle of the inviter.

Discussion

@method     plugInDidReceiveInvitation:forChatRoom:fromHandle:

When the IMServicePlugIn instance calls this method on the service application, Messages displays an invitation window from the specified handle for the chat room.

            If the user clicks "Accept" on the window, Messages then calls -joinChatRoom: on the
            service plug-in.  If the user clicks "Decline", Messages calls -declineChatRoomInvitation:
            instead.

- (oneway void)plugInDidReceiveNotice:(NSString *)notice forChatRoom:(NSString *)roomName

Parameters

notice

An informational message to be displayed.
roomName

The name of the chat room.

Discussion

@method     plugInDidReceiveNotice:forChatRoom:

When the IMServicePlugIn instance calls this method on the service application, Messages appends an information message to the transcript for the specified chat room.

            Since the IMServicePlugIn API does not support chat room properties such as topic,
            moderator status, or chat room settings; clients may use this method to display
            additional information which would otherwise be hidden to the user.

Parameters

Discussion

@method     plugInDidJoinChatRoom:

When the IMServicePlugIn instance calls this method on the service application, Messages indicates to the user that the specified chat room is now joined and ready to receive messages.

            This call should be used in response to joinChatRoom: being called on the service plug-in.

– plugInDidLeaveChatRoom:error: required method

@method     plugInDidLeaveChatRoom:error:
- (oneway void)plugInDidLeaveChatRoom:(NSString *)roomName error:(NSError *)error

Parameters

roomName

The name of the chat room.
error

An error, if any, which caused the user to be removed from the chat room, or prevented the user from joining the chat room.

Discussion

@method     plugInDidLeaveChatRoom:error:

The IMServicePlugIn should call this method on the service application to indiciate that a chat room has been successfully left, and that the user may attempt to re-enter that room in the future.

            This call may also be used in response to joinChatRoom: to indicate that an error occurred
            while joining the room.
