CNContactFormatter Class Reference

Inherits from NSFormatter
Conforms to NSSecureCoding
Declared in CNContactFormatter.h

Overview

This formatter handles international ordering and delimiting of the contact name components. This includes applying the user defaults when appropriate.

Parameters

Return Value

Discussion
