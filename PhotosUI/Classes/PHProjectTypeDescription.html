Overview

A PHProjectTypeDescription object represents one project type choice in the project picker that is presented to a user when creating a project via a project extension. An array of these objects is returned by the extenion in the [PHProjectExtensionController supportedProjectTypes] method. The info includes a type, localized title, description, image and optional subtype descriptions.
