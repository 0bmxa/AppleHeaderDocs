GKNoiseMap Class Reference

Inherits from NSObject
Declared in GKNoiseMap.h

Overview

GKNoiseMap represents an extracted portion of sampled points from continuous 3D noise. Extracted values are useful for 2D and 3D games. Noise values may be queried, set to explicit values or used as input for other uses, including textures and tile maps.

  size

The size of the 2D plane to extract from the 3D noise space, in noise space coordinates. Used together with origin to determine the bounds of the extracted plane. A smaller size captures a more “zoomed in” view of the noise, and vice versa.

@property (nonatomic, readonly) vector_double2 size
