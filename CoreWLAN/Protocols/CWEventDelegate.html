@protocol

@abstract Receive Wi-Fi event notifications.

– clientConnectionInterrupted

@method

- (void)clientConnectionInterrupted

Discussion

@method

@abstract Invoked if the connection to the Wi-Fi subsystem is temporarily interrupted.

@discussion All event notifications for which the Wi-Fi client has registered will be automatically re-registered if the connection is interrupted. The Wi-Fi client should re-sync any local state which is updated as a result of Wi-Fi event notifications.
