Overview

@class MPSCNNNeuronPReLU @dependency This depends on Metal.framework

Specifies the parametric ReLU neuron filter. For each pixel, applies the following function: f(x_i) = x_i, if x_i >= 0 = a_i * x_i if x_i < 0 i in [0…channels-1] i.e. parameters a_i are learned and applied to each channel separately. Compare this to ReLu where parameter a is shared acorss all channels. See https://arxiv.org/pdf/1502.01852.pdf for details.
